# 서비스 기획안

이 문서는 현재까지 확정된 정보만 기준으로 정리합니다.
아직 불확실한 내용은 추정으로 단정하지 않고 `미확정 사항`에 분리합니다.

## 서비스 개요

- 서비스 성격: 강남대학교 선후배 커피챗 서비스
- 현재 사용 서비스명: `KNU 커피챗`
  정식 확정 전까지 임시로 사용하는 이름
- 목적: 강남대학교 소속 사용자가 선후배 간 커피챗을 탐색하고 참여할 수 있는 웹 서비스

## 주요 사용자

- 강남대학교 소속 재학생
- 강남대학교 소속 휴학생
- 강남대학교 소속 졸업생

## 현재 라우트 기준 기능 범위

- `/`
  서비스 소개 및 주요 동선 진입

- `/login`
  로그인 페이지
  카카오 로그인, 구글 로그인 도입 예정

- `/my`
  마이페이지

- `/mentor/[id]`
  멘토 상세페이지

## 현재 우선순위

- 서비스 구조와 라우트 구조를 안정적으로 정리
- 로그인 연동을 위한 기본 구조 확보
- 마이페이지와 멘토 상세페이지를 실제 데이터 구조에 맞게 확장할 수 있도록 준비

## 미확정 사항

- 서비스명의 최종 확정 여부
- 백엔드 API 구조 및 인증 방식
- 백엔드 저장소와의 연동 방식
